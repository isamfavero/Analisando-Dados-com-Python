Analisando Dados com Python
Projeto de análise de dados desenvolvido em Python para identificar os principais fatores relacionados ao cancelamento de clientes. Utiliza tratamento de dados, análise exploratória e visualização gráfica para auxiliar na compreensão do comportamento dos clientes e apoiar tomadas de decisão estratégicas.


Stacks Utilizadas
O projeto foi desenvolvido utilizando a linguagem Python e executado no ambiente Jupyter Notebook. Para análise e manipulação dos dados, foi utilizada a biblioteca pandas. A leitura e manipulação de arquivos Excel foram realizadas com openpyxl. O ambiente interativo do notebook foi executado através do ipykernel, enquanto a biblioteca nbformat foi utilizada para suporte e manipulação da estrutura dos notebooks. Já as visualizações gráficas e análises interativas foram desenvolvidas utilizando Plotly.
